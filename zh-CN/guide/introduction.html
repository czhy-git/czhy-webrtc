CZHY WebRTC ​

基于 WebRTC & Janus API 封装设计的 Vue 组件库，开发体验良好，以优雅的方式完成 WebRTC 的接入。

：sparkles：特性 ​

  • 🚀 自动加载 Janus SDK，将 Janus 繁琐的 Api 封装进组件，你只需关注组件本身
  • 📦 20+ 高质量的开箱即用 Vue 组件以及 8+ hooks 封装
  • 📐 遵循直觉的、简约的 Api 设计
  • ⚡ Composition Api，更好的性能
  • 🔨 完整的 TypeScript 支持，更好的体验
  • 🧩 tree shaking 支持，模块分包，只打包你想要的的
  • 🚀 支持 volar，组件提供完善的代码提示

环境支持 ​

组件库 ​

CZHY WebRTC 可以在支持 ES2018 的浏览器上运行。如果您确实需要支持旧版本的浏览器，请自行添加 Babel 和相应的 Polyfill。

由于 Vue 3 不再支持 IE11，该组件库是基于 Vue3 封装，自然也不再支持 IE 浏览器。

Edge ≥ 79Firefox ≥ 78Chrome ≥ 64Safari ≥ 12
